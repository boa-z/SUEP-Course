# SUEP-Course

## Introduction

本项目是本人根据上海电力大学集成电路设计与制造专业的培养计划整理的课程资料，也包含了一些自己的学习笔记、作业和实验报告。

由于本人时间精力有限，难免会有一些错误，欢迎大家指正。也希望大家能够提交自己的学习资料，共同完善这个项目。

## 文件结构

```text
.
├── README.md
├── assets
├── notes # 笔记
├── codes # 各种代码
├── courseware # 课件
├── exams # 试卷
├── exercises # 习题
└── report # 实验报告
```

为了减小体积，所有课本和软件都不上传，只上传笔记、课程讲义和作业。
